This folder contains a portable source for desktop app.
To build, download the enviroment from here:
http://www.ultimatepp.org/

to add this project simply, create a file named "xmos_cpld_slice.var" in upp main folder (c:\upp):
UPP = "c:\\src\\github\\xmos_cpld_slice\\upp\\;C:\\upp\\uppsrc";
COMMON = "C:\\upp\\common";
OUTPUT = "C:\\upp\\out";
